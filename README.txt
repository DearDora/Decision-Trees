These programs programmed by Python 3.7
Simply run the .py program and it will generated the corresponding tree
The .pdf file will be in the same directory of the program

The program needs to use python package anytree by typing the following command:
pip install anytree

It also require Graphciz installed:
brew install python-graphviz

For T2 plotting, the program requires matplotlib and pandas with command:
pip install matplotlib
pip install pandas

The program programmed by MacOS system, used Wing101
